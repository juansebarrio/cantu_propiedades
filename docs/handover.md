## Tareas habituales

- Agregar un usuario nuevo
- Pausar/reactivar una propiedad

- Cambiar el mail de envío del reporte

## Si algo no funciona

- Reporte mensual no se envió → revisar `docs/runbook.md` o escribir a contacto@js80.studio
- Confirmación de WhatsApp no llega → ver troubleshooting (pendiente)
- App caída → status.vercel.com y status.supabase.com
